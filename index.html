Pestaña 1: Concepto e imagen

¿Qué es Cloud Computing?

Cloud Computing permite el acceso a recursos informáticos (servidores, almacenamiento y servicios) a través de Internet. Se utiliza para escalar servicios sin disponer de infraestructura física local.

¿Qué es SaaS, PaaS e IaaS?

SaaS entrega aplicaciones listas para usar por Internet; PaaS ofrece plataformas para desarrollar y desplegar aplicaciones; IaaS proporciona infraestructura virtual (máquinas y redes) bajo demanda.

¿Qué es Virtualización?

La virtualización crea instancias lógicas de hardware físico para ejecutar varios sistemas aislados en una sola máquina física. Facilita la gestión y optimización de recursos.

¿Qué es Docker?

Docker empaqueta aplicaciones y sus dependencias en contenedores ligeros y portables que se ejecutan de forma consistente en distintos entornos. Acelera el despliegue y la reproducibilidad.
